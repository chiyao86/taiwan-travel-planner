# 🗺️ 台灣旅遊規劃助手

一個簡單好用的台灣旅遊規劃工具，幫助您輕鬆規劃完美的台灣之旅！

## ✨ 功能特色

- 📍 **景點推薦** - 根據地區與類型推薦熱門景點
- 📅 **行程規劃** - 自訂天數與每日行程安排
- 💰 **預算估算** - 提供旅遊預算參考
- 🚗 **交通建議** - 根據行程提供交通方式建議
- 🌤️ **季節提示** - 針對不同季節的旅遊建議

## 🚀 使用方式

1. 在左側欄位設定旅遊資訊（日期、天數、地區）
2. 選擇旅遊類型與預算
3. 查看推薦景點與行程規劃
4. 參考旅遊建議與預算估算
